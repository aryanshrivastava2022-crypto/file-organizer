# File Organizer
A Python application that automatically organizes files into folders based on their file extensions.

# Features
- Organizes files automatically
- Creates folders based on file types
- Supports multiple file extensions
- Reduces folder clutter
- Easy-to-use command-line interface

# Requirements
- Python 3.x
  
# Modules Used
- os
- shutil

# How to Run
1. Download the project files.
2. Open the project folder.
3. Open a terminal or command prompt.
4. Run:
file_organizer.py
5. Enter the folder path you want to organize.

# Example
Before:
Downloads/
├── image.jpg
├── report.pdf
├── notes.txt
After:
Downloads/
├── JPG_Files/
│   └── image.jpg
├── PDF_Files/
│   └── report.pdf
├── TXT_Files/
│   └── notes.txt
